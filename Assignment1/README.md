# Assignment 1
Controlling the brightness of each channel of an RGB LED with the input given by 3 different potentiometers. 

## Requirements
- Arduino UNO
- Common Cathode RGB LED
- 3 potentiometers
- 2x100, 1x220 resistors (or higher)
- Wires
